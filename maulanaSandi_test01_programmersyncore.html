Jawab

1. Karena framework merupakan struktur kerangka berisi kode, dimana framework itu sendiri dapat membantu mempermudah dan mempersingkat waktu dalam pengembangan sistem.
2. if else merupakan percabangan, if berisikan pernyataan - mengeksekusi beberapa kode jika suatu kondisi benar dan kode lain jika kondisi tersebut salah,
while merupakan perulangan, mengeksekusi blok kode selama kondisi yang ditentukan benar.
while do sama dengan while namun disini kode dieksekusi dahulu baru terjadi perulangan.
switch case merupakan percabangan, pernyataan untuk memilih salah satu dari banyak blok kode yang akan dieksekusi dengan ekspresi dan program di tiap casenya,
for merupakan perulangan, digunakan ketika mengetahui berapa kali skrip harus dijalankan.
foreach merupakan perulangan untuk mengulang item-item dalam array.
3. consturctor adalah method khusus yang dieksekusi saat pembuatan objek
method sama seperti funsgi tapi method adalah kumpulan kode yang terkait dengan class
Class adalah sebuah pembungkus bisa berisi atribut,
variabel adalah nama untuk menyimpan nilai data,
parameter adalah variabel yang menyimpan nilai untuk sebuah fungsi,
procedure adalah pembungkus sebuha program agar dapat dipanggil berulang kali.
4. "Post" mengirimkan data langsung ke file tujuan dan tidak terlihat, jika "Get" mengirimkan data melalui URL dan dapat terlihat langsung pada URL tersebut.
5. sebuah sistem atau teknologi yang memungkinkan aplikasi dapat berkomunikasi dan berinteraksi dengan aplikasi lain melalui internet
6. MVC dan HMVC adalah pola arsitektur pengembangan. MVC fungsinya untuk memisah model, view, controller sedangkan HMVC sama namun terdapat Hierarki untuk memisahkan data tersebut.
7. curl() merupakan library untuk mengambil dan mengirimkan data melalui URL
8. untuk menampilkannya kita bisa menggunakan
var_dump($namafilearray);
9. perbedannya terletak pada protocolnya dengan pada HTTPS S nya adalah secure. HTTP tidak memiliki keamanan entkripsi data sedangkan HTTPS menyediakan SSL
10. API merupakan protokol yang memungkinkan satu aplikasi dapat berinteraksi dengan aplikasi lainnya. fungsinya sebagai penghubung
